## Description
Aplikasi ini dibuat untuk:
- Backend REST API ini berfungsi sebagai **penghubung antara website dan aplikasi mobile**.  
- API dibangun menggunakan **Django REST Framework**, dikemas dengan **Docker**, dan menggunakan **MySQL** sebagai database utama.
---

## Fitur Utama
- Autentikasi (JWT / Token)
- RESTful API
- Manajemen User
- Terhubung ke Aplikasi Mobile
- Terhubung ke Website Frontend
- Dockerized Environment
- Database MySQL
